Estimación de edad y genero con redes neuronales convolucionales.

Introducción

La estimación de edad y género es una taréa importante tanto para los humanos como para la la vision por computadora. Este tipo de estimación tiene muchas aplicaiones, principalmente para los forences y las redes sociales.

En esta entrada se muestran los resultados obtenidos para el problema de la estimación de edad y género utilizando una red neuronal convolucional utilizando keras.

Dataset utilizado

El dataset utilizado se llama UTKFace, es un dataset que consiste de un poco más de 20,000 imágenes con etiquedas de edad de entre 0 y 116 años, género y etnia.

Método utilizado

En la implementación se utiliza un método conocido como Deep EXpectation (DEX), en el cual se utiliza un modelo pre entrenado VGG-16.

Para realizar la clasificación, primero se realiza una detección de cara con algún método después se pasa por la red pre entrenada VGG-16 y por último este va a una softmax.

Para el error se utiliza el Mean Absolute Error (MAE), todos los resultados presentados son obtenidos utilizando este método. Este es un promedio del error absoluto entre la edad que se predijo y la edad verdadera.

Conclusiones

Las redes neuronales convolucionales son una opción bastante buena al momento de resolver el problema de clasificación de edad y género. Los resultados obtenidos en general fueron bastate satisfactorios a mi parecer.

El único inconveniente es que al momento de hacer el entrenamiento de la red, tuve que utilizar una base de datos mas pequeña de lo previsto, ya que el dataset original era el conocido como IMDB-Wiki dataset, el cual cuenta con más de 500, 000 imágenes. El problema fue que cuando intentaba pre procesar las imágenes se llenaba la ram de mi computadora y no podía pre procesarlas. El siguiente paso a realizar para este trabajo sería utilizar un dataset mas grande para así obtener aún mejores resultados, aunque los ya obtenidos a mi parecer no son malos para nada.
